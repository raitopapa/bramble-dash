# ブランブルのぼうけん（Bramble's Dash） 🌱

ブランブルたちが冒険する、**オリジナルIP**の横スクロール・アクションゲームです。
スーパーマリオ風の「ワールド制」で、タブレット・スマホのタッチ操作を最優先に作った、家族で遊べるキッズフレンドリーな作品。

- **対象端末**: iPad / iPhone / Android タブレット・スマホ（横向き推奨）、PCブラウザ
- **インストール不要**: `index.html` 1ファイルだけで完全に動作します
- **公開**: GitHub Pages（`main` / `(root)`）
- 姉妹作: [Bramble Boom!](https://github.com/raitopapa/bramble-boom)（ボンバーマン風）

---

## あそびかた

ゴールの旗をめざして進みます。敵は上から踏む（スター中は触れるだけ）、穴に落ちたり敵に当たるとミス。

- **4ワールド × （ステージ2＋ボス）＝全12ステージ**
  - 🌳 ワールド1 くさはら（陸・動く足場）
  - 🌊 ワールド2 みずべ（水中・泳ぐ／水流）
  - ☁ ワールド3 おおぞら（空・足場／風）
  - 🏰 ワールド4 まおうのしろ（洞窟→城・動く床）
- 各ワールドの最後に**専用ボス**（色・体力・攻撃がそれぞれ違う／弾・突進あり）
- 各ワールドのステージに**ボーナス土管**（入るとコイン部屋へワープ）

### キャラクター（最初から4人えらべる）

タイトルの ▲▼ でえらべます。それぞれ動きの個性があります。

| キャラ | とくちょう |
|---|---|
| 🟠 ブランブル | バランスがよい |
| 🟢 ミント | たかく ジャンプ |
| 🔵 ソラ | はやく はしる |
| 🩷 モモ | ふんわり おちる |
| ⭐ きんいろ（隠し） | ラスボス撃破で解放・全体強化 |
| 🌈 にじいろ（隠し） | ジェム4個で解放・さらに強化 |

### アイテム

| アイテム | こうか |
|---|---|
| コイン | 集める（スコア） |
| キノコ | 大きくなる |
| フラワー | ファイア（弾を撃てる・ボスの弾も消せる） |
| スター | むてき（敵も弾も触れて倒す） |
| はね（フライ） | 空をとべる |
| 💎 ジェム | 各ワールドに1個ずつ隠れている。4個集めると隠しキャラ解放 |

### そうさ

| | タッチ | キーボード |
|---|---|---|
| 移動 | 左下の十字キー | ← → / A D |
| ジャンプ | 右下のジャンプ | Space / Z / ↑ |
| ダッシュ／ファイア | ダッシュボタン | X / Shift |
| ポーズ | 右上 ❚❚ | P / Esc |

### むずかしさ（3段階）

ふつう ／ やさしい（5さい）／ とてもやさしい（3さい）。下げるほど、ジャンプや当たり判定がやさしくなり、
「とてもやさしい」はダメージやゲームオーバーがゆるくなります。

## オープニングとセーブ

- 起動すると **「はじめから」／「つづきから」** を選べます（セーブが無いときは「つづきから」は選べません）。
- **ステージをクリアすると進捗が自動保存**されます（解放状況・コイン・スコア・ジェム・選択キャラなど）。
- データはブラウザの localStorage（キー `brambleDash.save`）に保存されます。

---

## デプロイ（GitHub Pages）

1. リポジトリの **Settings → Pages → Deploy from a branch → `main` / `(root)`**
2. リポジトリ直下に **ビルド済みの `index.html`** があればそれだけで公開されます

> ⚠️ **重要**: 公開するのは必ず**ビルド済みの `index.html`**（単一ファイル・自己完結）です。
> `index.dev.html` は開発用なので、**これを `index.html` として置くとコードの文字がそのまま表示されます。**

---

## 開発

```bash
node build.mjs        # src/ → index.html（本番）と index.dev.html（開発）を生成
node test/smoke.mjs   # ヘッドレス自動テスト（実プレイ駆動＋ビルド成果物の起動確認）
```

- ソースの正は `src/`。**`index.html` は生成物なので直接編集しない**
- 設計と規約は [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md)、全数値の仕様は [`docs/SPEC.md`](docs/SPEC.md)、企画は [`docs/CONCEPT.md`](docs/CONCEPT.md)
- AIアシスタントで開発を続ける場合は [`CLAUDE.md`](CLAUDE.md) を最初に読ませてください
